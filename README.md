## Python Intern Projects

# Task 1: Hangman Game

-> Objective:
The objective of this project is to create a simple word-guessing game using Python.

-> Features

- Random word selection.
- Limited number of attempts.
- User-friendly gameplay.
- Displays progress after each guess.

-> Technologies Used

- Python
- Random Module
- Loops
- Conditional Statements
- Strings

-> Working

1. Select a random word from a predefined list.
2. Display hidden letters using underscores.
3. Ask the user to guess a letter.
4. Reveal correctly guessed letters.
5. End the game when the word is guessed or attempts are exhausted.

-> Sample Output

Word: _ _ _ _ _

Guess a letter: p

Word: p _ _ _ _

Guess a letter: y

Word: p y _ _ _

Congratulations! You guessed the word.

# Task 2: Stock Portfolio Tracker

-> Objective
The objective of this project is to create a simple stock portfolio tracker that calculates the total investment value based on user input.

-> Features

- Accepts stock name and quantity from the user.
- Uses a dictionary to store stock prices.
- Calculates total investment value.
- Displays the result to the user.
- Optionally saves the result in a text file.

-> Technologies Used

- Python
- Dictionary
- Input/Output Operations
- Basic Arithmetic
- File Handling

-> Working

1. Store stock prices in a dictionary.
2. Take stock name and quantity as input.
3. Find the stock price from the dictionary.
4. Calculate the total investment value.
5. Display the result.

-> Sample Output

Enter stock name: AAPL
Enter quantity: 5
Total Investment = 900

# Task 3: Task Automation with Python Scripts

-> Objective:
The objective of this project is to automate a repetitive task using Python.

-> Selected Task:
Move all JPG files from one folder to another folder automatically.

-> Features

- Scans a source folder.
- Identifies all JPG image files.
- Moves JPG files to a destination folder.
- Saves time by automating file management.

-> Technologies Used

- Python
- os Module
- shutil Module
- File Handling

-> Working

1. Specify source and destination folders.
2. Read all files from the source folder.
3. Check if a file has a ".jpg" extension.
4. Move the file to the destination folder.
5. Display a success message.

-> Sample Output

All JPG files moved successfully.

# Task 4: Basic Chatbot

-> Objective:
The objective of this project is to create a simple rule-based chatbot using Python.

-> Features

- Responds to predefined user messages.
- Uses conditional statements.
- Runs continuously until the user exits.

-> Technologies Used

- Python
- Functions
- Loops
- If-Else Statements
- Input/Output Operations

-> Working

1. Accept input from the user.
2. Compare the input with predefined commands.
3. Display the corresponding response.
4. Continue until the user enters "bye".

-> Sample Output

You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: bye
Bot: Goodbye!
